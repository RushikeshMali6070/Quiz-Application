Quiz Application

Project Description:
This is a web-based Quiz Application developed using HTML, CSS, and JavaScript.  
It allows users to attempt multiple quiz sheets with MCQs, timer functionality, scoring system, and voice feedback.

Academic Information:
This project was developed as part of my 3rd semester Project-Based Learning (PBL) course. 
It was developed to understand core concepts of web development.

Features:
- 5 Question Sheets
- 10 Questions per sheet
- MCQ based system
- Timer (Per question + Full test timer)
- Score calculation
- Voice feedback using Web Speech API
- Highlight correct and wrong answers

Technologies Used:
- HTML
- CSS
- JavaScript
- Web Speech API

How to Run
1. Download or clone the repository
2. Open `index.html` in browser
3. Select a sheet
4. Start the quiz

Learning Outcomes:
- Learned DOM manipulation in JavaScript
- Implemented timers using `setInterval`
- Used Web Speech API for voice interaction
- Designed responsive UI using CSS
